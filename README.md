# Library Sentinel

Library Sentinel is a toolkit for managing large music libaries.
Planned features are to discover and remove empty folders, find missing cue sheets, split single audio files into seperate files and metadata verification.

##
At this stage I have a working Scanner, Cue Finder, this shows how many Artists, Albums, Empty Albums and how many CUE Sheets there are.

##

- Scanner
- Spiltter
- Cue Hunter
- Libary Doctor
- Duplicate Finder
- Metadate Verification
