# MusicForge

MusicForge is a toolkit for managing large music libaries.

##

- Scanner
- Spiltter
- Cue Hunter
- Libary Doctor
- Duplicate Finder
- Metadate Verification

Missing artwork currently checks for external artwork files (.jpg, .jpeg, .png). Embedded artwork is planned for a future release.


Current Version:

v0.2.0(Cue Hunter)
